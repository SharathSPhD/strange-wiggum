AttractorFlow instruments your Claude Code agent using Finite-Time Lyapunov Exponents (FTLE) — a measure from dynamical systems theory:

λ < -0.05
CONVERGING
On track. Agent moving toward solution attractor.
λ > 0.25
DIVERGING
Off track. Trajectory unstable. Spawn explorer-agent.
MCP TOOLS AVAILABLE
attractorflow_record_state(text) # embed + record state
attractorflow_get_regime() # CONVERGING/STUCK/DIVERGING
attractorflow_get_lyapunov() # current λ value
attractorflow_checkpoint() # save good state
attractorflow_detect_bifurcation() # phase transition?
INSTALL
claude plugin install attractor-flow
USE IN YOUR AGENT PROMPT
"Call attractorflow_record_state() after every step.
Check get_regime() every 2-3 steps.
If DIVERGING → spawn explorer-agent to find new approach.
If CONVERGING → spawn convergence-agent to finish."
